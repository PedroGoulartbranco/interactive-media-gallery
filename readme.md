# 🖼️ Dynamic Media Gallery & Image Studio

A desktop application built entirely in **Python**, combining an interactive image gallery, an image editing studio, a customizable interface, multimedia controls, persistent local storage, and an integrated arcade game.

The project was developed from scratch using **Pygame** for the graphical interface and **Pillow (PIL)** for image processing, with user preferences and application data stored locally using **JSON**.

# ✨ Features

## 📂 Media Gallery

- Open and browse **any folder** on your computer.
- Automatic image discovery.
- Previous and next image navigation.
- Responsive image scaling while preserving aspect ratio.
- Save a favorite folder for automatic loading on startup.

---

## 🎨 Image Editing Studio

- Rotate images.
- Color adjustment filters.
- Negative (Invert/X-Ray) filter.
- Gaussian Blur.
- Combine multiple filters simultaneously.
- Automatically apply the current filter configuration to the next images.
- Export edited images with all applied effects.

---

## 🎨 Customizable Interface

- Fully responsive interface built entirely with **Pygame**.
- Adjustable window size.
- Custom interface colors.
- Persistent themes and settings.
- Keyboard shortcuts.
- Help and Settings menus.

---

## 💾 Persistent Data

The application automatically stores data using JSON files.

Saved information includes:

- Interface colors
- Window configuration
- Favorite media folder
- Image editing settings
- User preferences
- Romantic message board
- Balloon Pop leaderboard

---

## 💖 Message Board

A dedicated space for writing personalized messages.

Messages are automatically saved locally and restored whenever the application starts.

---

## 🎵 Multimedia System

- Background music.
- Multiple music tracks.
- Change music during runtime.
- Mute / Unmute controls.

---

## 🎮 Balloon Pop Arcade

A built-in arcade mini-game featuring:

- Pixel-perfect collision detection using Pygame masks.
- Floating score animations.
- Frame-rate independent gameplay.
- Automatic cleanup of temporary entities.
- Persistent Top 5 leaderboard stored in JSON.
